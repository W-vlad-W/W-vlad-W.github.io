== Installation ==

1. Upload the `ar-for-woocommerce.zip` to the `/wp-content/plugins/` directory and extract it.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the settings page in WooCommerce to get started.

== Frequently Asked Questions ==

= What 3D model formats are supported? =

AR for WooCommerce supports GLB, GLTF, USDZ, and Reality files for 3D model viewing. Other formats like DAE, DXF, 3DS, OBJ, PLY, STL, and PDF are also supported, and these files are automatically converted to GLB format.

= Does it support Android and iOS? =

Yes, the plugin supports AR on both iOS and Android. It uses the model-viewer scripts, launching native AR experiences on devices with WebXR (Android) and Quick Look (iOS).

= Are there any additional costs? =

The **free version** of the plugin allows for 1 model and basic features, while the **premium subscription** unlocks unlimited models and all advanced features. [Learn more about the premium subscription](https://augmentedrealityplugins.com).

= What if I don't have any 3D models? =

If you don't have your own 3D models, you can download free models from online libraries, create your own using mobile apps, or commission a 3D artist. Find out more in the [3D Model Resources](https://augmentedrealityplugins.com/support/#3d).
